## What This Project Does

PDFs made by different apps (Word, Google Docs, Python/ReportLab) have different
binary signatures. I convert the PDF binary data into grayscale PNG images and
train machine learning classifiers to figure out which app made each PDF.

- Label 0 = Word (Microsoft Word)
- Label 1 = Google Docs
- Label 2 = Python (ReportLab)

I train 4 classifiers (XGBoost, SVM, SGD, CNN) on the original images,
then test how well they hold up against 5 types of image augmentations.

## How to Run My Code

Install dependencies:
```bash
pip install -r requirements.txt
```

Run from the `ForensicsDetective/` folder, in this order:

```bash
# Step 1: Generate augmented images
python src/augmentation.py

# Step 2: Train all 4 classifiers at once
python src/classification.py

# Step 3: Run robustness analysis on augmented images
python src/analysis.py

# Step 4: Run McNemar's statistical significance tests
python src/statistical_tests.py
```

You can also train each classifier individually if needed:
```bash
python src/xgboost_classifier.py
python src/cnn_classifier.py
python src/svm_sgd_classifier.py
```

## My Results

### Accuracy on original images

| Model   | Accuracy |
|---------|----------|
| XGBoost | 99.89%   |
| SVM     | 99.55%   |
| SGD     | 99.11%   |
| CNN     | 99.66%   |

### Robustness across augmentations

| Augmentation | XGBoost | SVM    | SGD    | CNN    |
|--------------|---------|--------|--------|--------|
| Original     | 99.89%  | 99.55% | 99.11% | 99.66% |
| Gaussian     | 98.21%  | 99.55% | 98.99% | 99.66% |
| JPEG         | 98.21%  | 99.55% | 99.22% | 99.66% |
| DPI          | 86.13%  | 96.64% | 93.18% | 35.51% |
| Crop         | 48.10%  | 73.15% | 81.88% | 99.33% |
| Bit Depth    | 98.99%  | 99.66% | 99.11% | 26.29% |

### What I found
- SVM is the most robust overall (highest average accuracy across augmentations)
- XGBoost and SVM struggle with cropping because they use flattened pixel features
- CNN struggles with bit depth and DPI changes but handles cropping and noise well
- McNemar's test shows which classifier differences are statistically significant
